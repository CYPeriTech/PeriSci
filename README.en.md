# PeriSci

#### Description
开源近场有限元软件（英文名：PeriSci）是一款以近场动力学基有限元方法（简称近场有限元法，英文PeriFEM）为基础的、用于支持科学研究的开源CAE软件。该软件是近场有限元软件系统（PERISOFT）中的开源软件品牌。
