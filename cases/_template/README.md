## 字段标签协议说明

本 README 采用固定的编号标签协议，用于稳定解析与长期治理。

- 标签格式为：`[X]` 或 `[Xn]`
- 不同前缀表示不同语义分区
- 标签名称具有长期稳定性，不应被重新定义或改变语义
- 新增分区时，应增加新的标签前缀，而不修改既有前缀含义

当前保留的标签分区如下：

| 标签 | 含义                              |
| ---- | ---------------------------------|
| [I]  | 基本信息（Identity）              |
| [P]  | 问题定义（Problem）               |
| [A]  | 假设与适用范围（Assumptions）     |
| [C]  | 输入配置说明（Config）            |
| [R]  | 运行流程（Run）                   |
| [M]  | 回归判据（Metrics）               |
| [D]  | 数据集语义（Dataset Semantics）   |
| [S]  | 敏感性与稳定性（Sensitivity）     |
| [V]  | 可视化与展示建议（Visualization）  |
| [L]  | 局限性（Limitations）             |
| [E]  | 结果解释（Explanation）           |
| [X]  | 扩展与关联（Extensions）          |
| [H]  | 历史与版本演进（History）          |

> 若某分区不适用于本算例，可删除整个分区。
